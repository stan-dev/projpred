# Gaussian toy example

Gaussian toy example

## Usage

``` r
df_gaussian
```

## Format

A simulated regression dataset containing 100 observations.

- y:

  response, real-valued.

- x:

  predictors, 20 in total. Mean and SD are approximately 0 and 1,
  respectively.
