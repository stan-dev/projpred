# Mesquite data set

The mesquite bushes yields dataset from Gelman and Hill (2006)
(<https://sites.stat.columbia.edu/gelman/arm/>).

## Usage

``` r
mesquite
```

## Format

The response variable is the total weight (in grams) of photosynthetic
material as derived from actual harvesting of the bush. The predictor
variables are:

- diam1:

  diameter of the canopy (the leafy area of the bush) in meters,
  measured along the longer axis of the bush.

- diam2:

  canopy diameter measured along the shorter axis.

- canopy height:

  height of the canopy.

- total height:

  total height of the bush.

- density:

  plant unit density (# of primary stems per plant unit).

- group:

  group of measurements (0 for the first group, 1 for the second group).
